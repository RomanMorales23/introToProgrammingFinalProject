# introToProgrammingFinalProject

## Description

A top down 2d game where two player fight with projectiles.  

## Getting Started

### Dependencies

* Python
* Pygame Library
* Math Library
* os Library
* screeninfo Library
* sys Library
* time Library
* random Library 
* SETTINGS File

### Installing

* Download all files into one folder 
* Run MAIN file


### Executing program

* Execute the MAIN.py File
